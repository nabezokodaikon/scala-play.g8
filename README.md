# Scala with scalatest.
## Usage
### Master branch
```bash
$ g8 nabezokodaikon/scala-play
```

### Other branch
```bash
$ g8 nabezokodaikon/scala-play -b feature
```

## Test
```bash
$ g8 file://scala-play.g8/ --name=scala-play-test --force
$ cd scala-play-test
```
