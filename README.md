# Scala with scalatest.
## Usage
### Master branch
```bash
$ g8 nabezokodaikon/scala-basic
```
### Other branch
```bash
$ g8 nabezokodaikon/scala-basic -b feature
```
## Test
```bash
$ g8 file://scala-basic.g8/ --name=scala-basic-test --force
$ cd scala-basic-test
```
